# Projeto GameShop - Landing Page de Jogos

Este repositório contém o desenvolvimento de uma página de vendas para uma loja de games, focada em design moderno, responsividade e interatividade.

## O que foi realizado:
* **Interface Temática:** Desenvolvimento de um layout focado na experiência do usuário gamer.
* **Vitrine de Produtos:** Estruturação de uma grade de jogos com detalhes de preço e especificações.
* **Interatividade com jQuery:** Implementação de efeitos visuais e navegação suave entre seções.
* **Layout Responsivo:** Uso de CSS moderno para garantir a compatibilidade com smartphones e tablets.

## Tecnologias:
* HTML5
* CSS3 / SASS
* JavaScript / jQuery
* Git / GitHub
